# Volumes

CSI = Container Storage Interface

## Volume Drivers

Volume drivers allow you to create volumes. For instance: Azure File Storage.

## Volume with Host Path

```
apiVersion: v1
kind: Pod
metadata:
  name: test-pd
spec:
  containers:
  - image: k8s.gcr.io/test-webserver
    name: test-container
    volumeMounts:
    - mountPath: /test-pd
      name: test-volume
  volumes:
  - name: test-volume
    hostPath:
      # directory location on host
      path: /data
      # this field is optional
      type: Directory
```
